# Phase 09.5 Plan 02: Shell Admin — Sidebar Colapsável, Topbar Branca, Layout Dinâmico

**One-liner:** Sidebar admin colapsável (240px ↔ 60px) com toggle, user pill + sign-out no footer, topbar branca com breadcrumb derivado por rota, e margin-left dinâmico sincronizado entre sidebar e área principal.

## O Que Foi Construído

### Task 1: AdminSidebar — toggle + colapso + user pill

**`AdminSidebar.tsx`** completamente refatorado:

- **Toggle button absoluto** (`top-3 -right-3`) com ChevronLeft/ChevronRight de 14px e fundo branco — clica para alternar entre 240px e 60px
- **Transição suave:** `transition-[width] duration-[250ms] ease-out` — apenas width transiciona, sem recalcular outros props
- **Labels condicionais:** cada `<span>` de texto e badge "Em breve" recebe `cn('whitespace-nowrap', collapsed && 'hidden')` — desaparecem quando colapsado
- **User pill no footer:** avatar circular com inicial + email truncado + botão "Sair" quando expandido; apenas avatar com `title` tooltip quando colapsado
- **Sign-out handler:** `handleSignOut` com `try/catch` e `toast.error` — migrado do AdminHeader
- `useSidebarCollapsed` importado do hook do plan 01 como single source of truth

### Task 2: AdminHeader — topbar branca + breadcrumb + slot de actions

**`AdminHeader.tsx`** transformado de componente stateful em componente apresentacional puro:

- **Props novas:** `AdminHeaderProps { breadcrumb: BreadcrumbItem[], actions?: ReactNode }`
- **Interface exportada:** `BreadcrumbItem { label: string, href?: string }` — usada pelo AdminLayout
- **Topbar branca:** `bg-white border-b border-gray-200 sticky top-0 z-20 h-14` com shadow sutil
- **Breadcrumb renderizado:** Fragment com separador `›`, Link para itens navegáveis, span com `aria-current="page"` para o item ativo
- **Acessibilidade:** `aria-label="Breadcrumb"` na `<nav>` + `aria-current="page"` no último item
- **REMOVIDOS:** `useAuth`, `signOut`, `handleSignOut`, `Button`, `useNavigate`, `useToast`, JSX de email e "Encerrar sessão"

### Task 3: AdminLayout — margin-left dinâmico + breadcrumb por rota

**`AdminLayout.tsx`** refatorado para coordenar sidebar e header:

- **useSidebarCollapsed** importado — estado readonly `[collapsed]` para derivar margin-left
- **Margin-left dinâmico:** `cn('...transition-[margin-left]...', collapsed ? 'ml-[60px]' : 'ml-60')` — acompanha a sidebar em 250ms
- **Breadcrumb derivado por rota:**
  - `/admin/orgs/:orgId` → `Admin › Organizações › Org {orgId.slice(0,8)}`
  - `/admin/dashboard` → `Admin › Organizações`
  - fallback → `Admin`
- `useMemo` nos deps `[orgDetailMatch, location.pathname]` — recalcula apenas quando rota muda
- **REMOVIDO:** `mt-14 ml-60 p-8` estático no `<main>` — `sticky` no header + `flex-1` no main cuidam do layout
