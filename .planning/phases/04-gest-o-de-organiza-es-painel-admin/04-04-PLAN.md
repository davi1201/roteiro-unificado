<objective>
Implementar a página de detalhe da organização e o fluxo de adição de membro via Edge Function segura.
A criação de usuário requer `service_role` key — que NÃO pode estar no browser. Solução (D-01, D-02, D-03):

então faz INSERT em `org_members` com o user_id retornado + role='company'.

Purpose: ORG-03 — admin precisa criar usuários para as construtoras. Fluxo seguro: service_role nunca
sai do servidor; cliente faz apenas INSERT em org_members (autorizado por RLS is_admin()).

Output: Página /admin/orgs/:orgId funcional com listagem de membros e modal de convite operacional.
</objective>

    ```
      'Access-Control-Allow-Origin': '*',
    ```

    Handler:
    ```
    Deno.serve(async (req: Request) => {
      // CORS preflight
      if (req.method === 'OPTIONS') {
        return new Response('ok', { headers: corsHeaders })
      }

      if (req.method !== 'POST') {
        return new Response(
          JSON.stringify({ error: 'Method not allowed' }),
          { status: 405, headers: { ...corsHeaders, 'Content-Type': 'application/json' } }
        )
      }

      try {
        const body = await req.json().catch(() => null)
        if (!body || typeof body !== 'object') {
          return new Response(
            JSON.stringify({ error: 'Invalid JSON body' }),
            { status: 400, headers: { ...corsHeaders, 'Content-Type': 'application/json' } }
          )
        }

        const { email, password, org_id } = body as { email?: unknown; password?: unknown; org_id?: unknown }

        // Validação de input — três campos string não-vazios
        if (typeof email !== 'string' || !email.match(/^[^@\s]+@[^@\s]+\.[^@\s]+$/)) {
          return new Response(
            JSON.stringify({ error: 'Invalid email' }),
            { status: 400, headers: { ...corsHeaders, 'Content-Type': 'application/json' } }
          )
        }
        if (typeof password !== 'string' || password.length < 8) {
          return new Response(
            JSON.stringify({ error: 'Password must be at least 8 characters' }),
            { status: 400, headers: { ...corsHeaders, 'Content-Type': 'application/json' } }
          )
        }
        if (typeof org_id !== 'string' || !org_id.match(/^[0-9a-f]{8}-[0-9a-f]{4}-[0-9a-f]{4}-[0-9a-f]{4}-[0-9a-f]{12}$/i)) {
          return new Response(
            JSON.stringify({ error: 'Invalid org_id' }),
            { status: 400, headers: { ...corsHeaders, 'Content-Type': 'application/json' } }
          )
        }

        const { data, error } = await adminClient.auth.admin.createUser({
          email,
          password,
          email_confirm: true,
        })

        if (error || !data.user) {
          return new Response(
            JSON.stringify({ error: error?.message ?? 'Failed to create user' }),
            { status: 400, headers: { ...corsHeaders, 'Content-Type': 'application/json' } }
          )
        }

        return new Response(
          JSON.stringify({ user_id: data.user.id }),
          { status: 200, headers: { ...corsHeaders, 'Content-Type': 'application/json' } }
        )
      } catch (_err) {
        return new Response(
          JSON.stringify({ error: 'Internal server error' }),
          { status: 500, headers: { ...corsHeaders, 'Content-Type': 'application/json' } }
        )
      }
    })
    ```

    Notas:
    - Validação de email regex simples (não algoritmo RFC completo — suficiente para piloto).
    - Validação de org_id como UUID v4-ish (regex de 8-4-4-4-12 hex chars).
    - Senha mínimo 8 caracteres (alinhado com UI-SPEC §Add member labels).
    - `email_confirm: true` faz user já estar verificado — pode fazer login imediatamente (D-02).
    - NÃO faz INSERT em `org_members` — esse passo é responsabilidade do cliente (D-03) e respeita RLS is_admin().

    Hook:
    ```
    export function useOrgDetail(orgId: string | undefined) {
      const orgQuery = useQuery<Tables<'orgs'>>({
        queryKey: ['orgs', orgId],
        queryFn: async () => {
          const { data, error } = await supabase
            .from('orgs')
            .select('id, name, cnpj, active, created_at')
            .eq('id', orgId!)
            .single<Tables<'orgs'>>()
          if (error) throw error
          return data
        },
        enabled: !!orgId,
      })

      const membersQuery = useQuery<Tables<'org_members'>[]>({
        queryKey: ['org_members', orgId],
        queryFn: async () => {
          const { data, error } = await supabase
            .from('org_members')
            .select('id, org_id, user_id, role, created_at')
            .eq('org_id', orgId!)
            .order('created_at', { ascending: false })
          if (error) throw error
          return data ?? []
        },
        enabled: !!orgId,
      })

      return {
        org: orgQuery.data,
        members: membersQuery.data,
        isLoading: orgQuery.isLoading || membersQuery.isLoading,
        isError: orgQuery.isError || membersQuery.isError,
        error: orgQuery.error || membersQuery.error,
      }
    }
    ```

    ```
        // Step 1 (D-01): invocar Edge Function — service_role isolada no Deno runtime
        const { data: fnData, error: fnError } = await supabase.functions.invoke<{ user_id: string }>(
          'create-user',
          { body: { email: data.email, password: data.password, org_id: orgId } }
        )
        if (fnError || !fnData?.user_id) {
          // Mensagem específica se email já existe (Edge Function devolve 400 com error.message do supabase auth)
          const message = fnError?.message ?? ''
          if (message.toLowerCase().includes('already') || message.toLowerCase().includes('exists')) {
            throw new Error('email_exists')
          }
          throw new Error('edge_function_failed')
        }

        // Step 2 (D-03): INSERT em org_members com user_id retornado
        const { error: memberError } = await supabase.from('org_members').insert({
          org_id: orgId,
          user_id: fnData.user_id,
          role: 'company',
        })
        if (memberError) throw new Error('org_members_insert_failed')
      },
      onSuccess: () => {
        queryClient.invalidateQueries({ queryKey: ['org_members', orgId] })
        queryClient.invalidateQueries({ queryKey: ['orgs'] })  // member_count desatualiza
        toast.success('Membro adicionado com sucesso')
        reset()
        onClose()
      },
      onError: (err: Error) => {
        if (err.message === 'email_exists') {
          toast.error('Este email já está cadastrado no sistema.')
        } else {
          toast.error('Não foi possível adicionar o membro. Verifique o email.')
        }
      },
    })

    const onSubmit = (data: AddMemberFormData) => addMemberMutation.mutate(data)
    ```
