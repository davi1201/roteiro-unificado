# Sketch Manifest

## Design Direction
Layout admin limpo e funcional para uso interno do time SuaEquipe.IA. Sidebar sempre visível é prioridade. Formulário com alta densidade de informação deve ser navegável sem scroll excessivo. Vibe: profissional/operacional, não marketeira — pensa em ferramenta de trabalho, não landing page.

## Reference Points
- Tailwind stack existente (bg-primary = azul escuro, accent = amarelo)
- Sidebar padrão de admin dashboards SaaS (Vercel, Linear, Notion)
